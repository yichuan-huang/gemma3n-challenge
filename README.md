# Garbage Classification System

A Gradio app for classifying garbage into 4 categories using Gemma-3n model.

## Usage
1. Upload an image.
2. Click "Classify Garbage" or wait for auto-classification.

## Categories
- Recyclable Waste
- Food/Kitchen Waste
- Hazardous Waste
- Other Waste
