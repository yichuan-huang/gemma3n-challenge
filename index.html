The Technical Details

What we're working with:

  • Model: Gemma 3n E4B-it (4 billion parameters, efficient memory usage)
  • Image Handling: Resizes to 512x512 while keeping proportions right
  • Prompting: Carefully crafted questions that get reliable answers
  • Response Parsing: Smart text analysis with backup plans
  • Safety Features: Filters out inappropriate content automatically

Problems We Hit (And How We Fixed Them)

🔴 What Went Wrong

  • Apple Silicon Macs didn't play nice with the GPU code
  • First model version was pretty inaccurate
  • Images got stretched and looked weird
  • Model responses were inconsistent and hard to parse
  • Struggled with mixed trash scenarios
  • Needed to handle non-waste images safely

🟢 How We Solved It

  • Added smart CPU fallback for Mac users
  • Upgraded to the much better E4B model
  • Fixed image processing to preserve aspect ratios
  • Built robust parsing with multiple backup strategies
  • Added contamination detection with clear rules
  • Created comprehensive safety filters

What We Achieved

~7s Processing Time (H200)
5-7GB Memory Usage
Good Accuracy

What it can do now:

  • Works completely offline - no internet connection required
